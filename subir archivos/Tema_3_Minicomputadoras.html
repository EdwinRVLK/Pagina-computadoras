Minicomputadoras

¿Qué son?

Una minicomputadora no es como su nombre puede indicar, una computadora muy pequeña.
Debe su nombre a cuando las computadoras eran enormes, y cualquier desarrollo computacional menor a los tamaños acostumbrados en esos días, debía ser llamado “mini”.
Podría decirse que las minicomputadoras son un tipo específico de computadora multiusuario, ubicadas, en términos de proceso computacional, entre los grandes sistemas multiusuario del tipo mainframe y las computadoras, estaciones de trabajo y sistemas de datos monousuario, como nuestra propia PC.

Sin embargo, a pesar de estar emparentadas con los mainframes por capacidad, lo cierto es que no ofrecen el mismo nivel de procesamiento.
Nacen en la década de 1960 de la necesidad de contar con un sistema de cómputo de bajo costo, comparándolo con un mainframe, y de bajo mantenimiento, tanto de especialistas para operarlo como de personal técnico para repararlo.


Para qué sirven?

La implementación de las minicomputadoras varían de acuerdo a al sector en el cuál se esté utilizando. En este sentido, desde fábricas y empresas, hasta laboratorios científicos y universidades han dado uso a las minicomputadoras. A razón de esto, a continuación se explican los principales usos de las minicomputadoras.
Para tareas funcionales:
Las minicomputadoras eran utilizadas para distintas áreas empresariales, siendo capaces de realizar múltiples tareas. Así, se emplearon las primeras minicomputadoras para establecer la informática departamental dentro de organizaciones de mayor volumen. Otra tarea interesante que se les daba a las minicomputadoras, era la de crear redes de minicomputadoras que lograban funcionar mejor que una sola macrocomputadora.

Para programación:

Uno de los usos más importantes que se le dió a las minicomputadoras, fué en el área de la programación. Esto hizo posible que a un menor costo de equipos, más usuarios tuvieran la posibilidad de involucrarse en la programación. Siendo así la forma en la que se amplió la tasa de innovación de software, además, esto contribuyo a mejorar la interfaz de usuarios para la informática personal.

Para controlar procesos:

El uso de minicomputadoras también se extendió a empresas de control de proceso de fabricación, quienes realizaban diseños de ingeniería, experimentos científicos y sistemas de comunicación. Estas compañías, por lo general, necesitan la adquisición de datos y retroalimentación del mismo. De esta manera, las minicomputadoras se encargaban de cumplir esas dos funciones para mejorar la fluidez y el control de esos procesos.

Para gestionar datos:

Las minicomputadoras eran capaces de gestionar datos y cualquier tarea relacionada a ello. Así, funciones como tomar datos, restaurarlos, e incluso generar datos, eran tareas bastante útiles que estos equipos realizaban durante su época.

Para la comunicación:

El software es lo que permite la interacción entre el usuario y el ordenador. En este sentido, las minicomputadoras actúan como interfaz entre el usuario y un procesador mayor, dando como resultado que una minicomputadora le permita al usuario poder comprobar errores o realizar ajustes del mismo.
